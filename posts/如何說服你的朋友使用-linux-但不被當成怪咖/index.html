如何說服你的朋友使用 Linux - 但不被當成怪咖
2011-11-02 03:15 label Linux   翻譯  

本文翻譯自「How to Convince Your Friends to Use Linux Without Being a Jerk」，作者為 Abhishek。

This article is a translation of “How to Convince Your Friends to Use Linux Without Being a Jerk”, which is written by Abhishek.

Linux 是目前世界上最安全穩定的系統之一，但它的使用者數量卻不如想像中的成長迅速。這當中有很多原因，不過我們先不討論。但如果你像其他的 Linux 使用者一樣，對現在 Linux 的市佔率感到失望，我們可以告訴你一些說服你朋友來用 Linux 的小技巧。

現在有很多 Linux 使用者試著去引誘他們的朋友或家人來嘗試一個有名且容易上手的發行版 - Ubuntu。有些人成功了，不過大部分是失敗的。所以，我們要告訴你一些技巧來幫助你推廣 Linux，但不會被視為一個自以為是又看不起 Windows 或 Mac 的電腦怪咖。

用秀的！不要用說的。

沒錯，如果你想要說服一個 Windows 或 Mac 的使用者來使用 Linux，這是第一件也是最重要的一件事。Ubuntu、Linux Mint與許多其他發行版看起來都相當漂亮，而且說真的，最新版的 Ubuntu (Ubuntu 11.10) 看起來就跟 Mac 一樣漂亮。不過，如果你只是跟他們用說的，你認為他們會同意你嗎？就算他們同意了，他們還是不清楚 Ubuntu 或是其他你說的發行版究竟長什麼樣子。我的建議是，你可以帶著你的 Linux 筆電，交給對方讓他們玩玩看。如果你用的是 Ubuntu，我建議你開一個新的 Guest Session (訪客使用者階段)，然後把電腦交給他。這樣一來，他們會更了解一個沒有經過調整的桌面有多漂亮。如果你不太可能向你的朋友展示桌面的話，那就傳個關於桌面展示的 Youtube 影片給他吧。

這個「用秀的不要用說的」作法也可以用在功能比較上。假設你想要描述 Linux 一些明顯的優點，不要跟他囉嗦一堆不會中毒、沒有磁碟碎片、或是桌面效能有多快之類的話，只要秀給他們看就好了。把他們的 Windows/Mac 筆電拿來放在旁邊，開始計時看他們開機有多快。我不想再說一遍，用秀的，不要用說的。

別再告訴他們 Windows 不好，他們已經知道了

對一個想要把 Windows 使用者帶向光明的 Linux 使用者來說，他們總是把「Windows 很爛」這件事掛在嘴邊。至少很多 Linux 使用者都這樣想。抨擊 Windows 很好，大家都在罵這個討厭的作業系統，但是這對 Windows 使用者而言沒什麼好說的，我很確定他們都已經知道了。我相信當你在閱讀這篇文章的時候，至少有上千個 Windows 使用者在痛罵微軟。不過他們沒辦法改變什麼。

簡單來說，不要告訴他們 Windows 不好，甚至不要提到 Windows。只要告訴他們：「嘿，我的電腦桌面超棒的，你該試試看。」把電腦給他們，然後讓他們去痛罵 Windows。當我把我的朋友變成 Linux 使用者的時候，我所做的就只是給他們看我的電腦，沒有提到關於 Windows 或 Mac 的隻字片語。當他們看到我的電腦的時候，他們就開始瘋狂地大罵 Windows 了。

如果他們遇到問題

與其直接幫助他們，不如告訴他們怎麼在 Linux 上取得幫助。別讓人依賴你解決所有他們電腦上的問題。只要帶他們看過 Ubuntu 論壇、IRC 以及說明文件，讓他們自己找到答案就好。

不要強調「free」的部份

我再說一次，不要。不要推銷「free」這個部分。事實上，你最好完全不要提到免費這件事。有時候別人會對於 Linux 印象深刻，所以他們可能會問你它要花多少錢。你可以輕鬆的看著他們，然後說「It’s free.」接著等待他們的回應就好。

譯註：

  1. 外觀的好壞很主觀，我並不認為 Ubuntu 的 artwork 有跟 Mac 一樣漂亮….
  2. 操作性(使用者體驗)而言，我認為 Mac 還是優於目前我所用過的幾種 Linux 桌面。但還是有些地方我比較喜歡 Linux 上的設計。
